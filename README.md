# Torchy Baselines

PyTorch version of [Stable Baselines](https://github.com/hill-a/stable-baselines), a set of improved implementations of reinforcement learning algorithms.

## Implemented Algorithms

- CEM-RL (with TD3)
- PPO
- SAC
- TD3
